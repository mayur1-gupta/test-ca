Instruction

The memory card game is a simple and fun game for single player. It is played on a 4x4 grid.player goal is to find the same images in cards.

Typically, player start the game, In game screen, player can see 16 cards, player first flip any two cards if both cards have different image then cards flip and if both cards have same image then both cards images are shown.

If player can successfully open all the cards then player win the game.
